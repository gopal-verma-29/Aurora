The full pipeline
Browser → POST /api/leads with niche, city, filters, language → Google Maps text search (3 pages, 60 raw) → filter by your rating + review thresholds → shuffle for variety → Place Details for phone + website → Claude with niche-specific prompt + language instruction → ranked lead cards with copy-as-text.

Copy as text
Each lead has a "copy as text" button. It formats the lead as a clean WhatsApp-ready block — name, phone, rating, address, pitch. Paste directly into WhatsApp or any CRM.

Shortlist persistence
Saved leads write to shortlist.json on disk + leads.xlsx for Excel. Server restarts won't lose your shortlist. Excel file records the pitch language used.
